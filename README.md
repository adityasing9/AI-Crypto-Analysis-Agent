## 🧠 Overview

AI Crypto Analysis Agent is a **multi-model intelligent system** that analyzes cryptocurrencies using real-time market data and AI reasoning.

It compares multiple AI model outputs and selects the best response using a scoring engine.

## ⚙️ Setup

```bash
git clone https://github.com/YOUR_USERNAME/ai-crypto-agent.git
cd ai-crypto-agent
pip install -r requirements.txt
streamlit run app.py
```
